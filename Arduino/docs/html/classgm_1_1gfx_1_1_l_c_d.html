◆ setLR()

void gm::gfx::LCD::setLR ( )

Unlike the 932X drivers that set the address window to the full screen by default (using the address counter for draw_px operations), the 7575 needs the address window set on all graphics operations. In order to save a few register writes on each pixel drawn, the lower-right corner of the address window is reset after most fill operations, so that draw_px only needs to change the upper left each time.
